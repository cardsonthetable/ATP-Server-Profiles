# ATP Server Profiles
In this project, I apply Principal Component Analysis to five serve metrics — ace rate, double-fault rate, first-serve percentage, first-serve points won, and second-serve points won — to identify the underlying dimensions of serve ability. The two principal components (which I define as Serve Dominance and Serve Dependability) explain 76.6% of the variance in serve profiles across players and an OLS regression on those components explains 44% of the variation in match win percentage. The coefficient on Serve Dependability has a larger magnitude which suggests that consistency matters more than aggression on the ATP Tour.

This was originally from a Machine Learning class project in the Spring of 2025. As a tennis fan interested in the game's advanced analytics, I used match-level data from Jeff Sackmann's ATP repository to analyze serve performance across the modern tour. I've brushed up the original file and uploaded it here.
